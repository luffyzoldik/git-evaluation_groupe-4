## Installation

Il va falloir utiliser VSC https://code.visualstudio.com/
Il va falloir utiliser un terminal Ubuntu (wsl est le plus simple)
Afin d'éxécuter les différents scripts il faudra également une version de python la meilleure est la dernière

## Exécution

Ouvrez votre VSC, puis un termianl en wsl (Ubuntu)
Vous pouvez désormais éxécuter une commande afin de tester les différents scripts
echo "3*9" | python3 minitrice
echo 3*+9" | python3 minitrice 
python3 generator 2 | python3 minitrice
